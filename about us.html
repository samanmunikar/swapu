Technology

We write codes, scale systems and validate applications. We work in multiple platforms to create intelligent systems based on Natural Language Processing, Predictive analytics, Machine learning technology. The tech team is categorized, based on the platforms they work with and the functions they perform as:

  • Frontend

    We are responsible for visualizing and designing the interactive side of web applications and building libraries. We mostly work with Ember JS to develop AI tools and typically work with back-end engineers along with the QA team to ensure that the application meets all usability requirements.

  • Backend

    We design and develop everything that goes under the hood. We primarily use Java and related technologies like Spring framework, Gradle, Tomcat, EJB, Hibernate, Hadoop, and ElasticSearch. The databases that we use primarily includes MySQL, MongoDB and Redis. We follow the best practices to build RESTful web services for our applications. We specialize in Machine Learning and Natural Language Processing which power most of our systems.

  • QA

    We screen codes and scan for irregularities in all our software and processes. Through a series of usually recursive testing using software like Eclipse (Selenium Web-Driver for browser automation) and Selenium IDE, we make sure that all our products are bug free. Only validated products are declared QC passed and fit for release.
